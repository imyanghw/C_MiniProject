# C_MiniProject
비디오 대여 관리 프로그램
> 파일 포인터 함수를 사용하여 사전에 등록한 데이터들을 읽어와 프로그램을 동작


## 개발환경 및 사용버전
- Visual Studio 2017
- C Language

## 구현내용
- 신규 고객 등록
- 고객 정보 조회
- 전체 고객 정보 조회
- 신규 비디오 등록
- 비디오 정보 조회
- 전체 비디오 정보 조회
- 대여, 반납, 대여현황 조회
- 시간함수 Epoch times를 사용하여 현재시간과 날짜 정보를 받아옴(초단위로 환산하여 시간을 산출하였음)
